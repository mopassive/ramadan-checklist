## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
npm start
```

Visit [http://localhost:3000](http://localhost:3000) to view the app.

## Checklist Categories

1. **Salah** - Daily prayers and Taraweeh
2. **Fasting** - Daily fast tracking
3. **Qur'an & Learning** - Reading and study
4. **Du'a & Protection** - Supplications and remembrance
5. **Dhikr** - Various forms of remembrance
6. **Character** - Personal conduct
7. **Charity & Family** - Sadaqah and helping parents

## Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first styling
- **localStorage** - Client-side persistence
